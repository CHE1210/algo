# 문제 제목: 세수정렬

## 문제 정보
**출처:** noj.am/2752
**난이도:** Easy, Medium, Hard 중 하나.

## 문제 설명
동규는 세수를 하다가 정렬이 하고 싶어졌다.

정수 세 개를 생각한 뒤에, 이를 오름차순으로 정렬하고 싶어졌다.

정수 세 개가 주어졌을 때, 가장 작은 수, 그 다음 수, 가장 큰 수를 출력하는 프로그램을 작성하시오.

## 해결 과정

### 접근 방법
list 활용
## 코드
```python
a, b, c = map(int, input().split())

list = [a, b, c]
list.sort()

print(list[0], list[1], list[2])
